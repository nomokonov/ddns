REST приложение для огранизации DynDNS
на базе сервиса от  Яндекс почта для домена и его API
https://yandex.ru/support/pdd/api.html

Такая схема нужна была т.к. сам сервис распологался за пределами сети для которой определяется IP
иначе определение IP можно  было организовать в самом приложении.
Запросы на сервис отправляет Mikrotik из сети lkz которой назначется IP
